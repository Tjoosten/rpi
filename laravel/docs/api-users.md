[API] Users: 
==========================

## Endpoints

The user part of the APi has the following endpoints.

| Enpoint       | `GET` | `POST`| `PATCH`| `DELETE` |
| ------------- | ----- | ----- | ------ | -------- |
| `/user/all`   | YES   | -     | -      | -        |
| `/user/insert`| -     | YES   | -      | -        |
| `/user{id}`   | YES   | -     | -      | YES      |

## Headers

The API support the following HTTP Header: 

| ------ | ------------------ |
| `JSON` | `application/json` |
